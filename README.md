# ❄️Coca Cola - Scroll Into The Chill
A cinematic Coca-Cola inspired scroll animation experience buitlt using only ***HTML and CSS***

---

## ✨ Features
- Scroll-driven text transitions
- Warm-to-cold Coca-Cola bottle transformation
- Animated ice cube motion from both sides
- Cinematic blue and opacity effects
- Pure HTML and CSS implementation
- Smooth scroll timeline animations

----

## 🧊 Animation Flow
### Intro
- *SCROLL DOWN* indicator
- Cinematic text reveal

### Mid Section
- Ice cubes enter from left and right
- Coca-Cola bottle transforms into a chilled version

### Final Section
- Final tagline reveal
- Fully frozen visual atmosphere

----

## 🛠️ Built With
- HTML
- CSS
- CSS Scroll Timeline Animations
- CSS Keyframes
- CSS Transform and Blue Effects
